# Guardas

Las guardas son esta feature que nos permite decir "esta funcion se ejecuta con esta entrada"
igual que en las definiciones de funciones partidas en matematica

$$
f(x) = 
\begin{cases}
 1 & x <= 0 \\
 x f(x - 1) & x > 0 \\
\end{cases}
$$

Lo cual es muy positivo ya que nos deja ver la programacion muy alineada a la matematica y nos
permite hacer demostraciones matematicas formales sobre nuestras funciones
